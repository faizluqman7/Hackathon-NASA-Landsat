    • What is Landsat Observer?
        Background
        • Landsat Observer was created to provide easy access to satellite data for researchers and hobbyists interested in Earth observation.
        Tech Stack
        • The application is built using:
          • CSS/HTML/JS
            • 98.css library by jdan
          • Python
            • json5
            • rasterio
            • python-dotenv
          • Landsat Data
            • Machine-to-Machine (M2M) API by USGS
            • Landsat Data provided by NASA
